# AndroidRefreshLayout
Android万能刷新加载框架，适用于所有View，可根据自己需求定制！<br>
本Demo演示如何使用通用的下拉刷新上拉加载控件，demo中已经实现了常见的需要上下拉功能的控件，其他控件如果需要加入这两个功能可自行扩展，实现Pullable接口即可
